\# To-Do List Application



A simple Django-based To-Do List web application that allows users to manage daily tasks.



\## Features

\- Create new to-do items

\- View to-do details

\- Edit existing to-do items

\- Delete to-do items



\## To-Do Fields

\- Subject

\- Notes



\## Modules

\### Module 1: To-Do Creation \& Listing

\- Add new to-do items

\- Display all to-dos on the home page



\### Module 2: To-Do Update \& Deletion

\- View and edit existing to-dos

\- Delete to-do items



\## Technologies Used

\- Python

\- Django

\- HTML

\- CSS

\- Git \& GitHub

python manage.py migrate
